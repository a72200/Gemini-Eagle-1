## 🚀 How to Contribute
1. **Fork the Repository:** Create your own branch from `main`.
2. **Local Testing:** Run `docker-compose up` to test your environment locally before pushing to Google Cloud.
3. **Submit a Pull Request (PR):** Describe your enhancements clearly (e.g., adding features to the Law, Architect, or Ambassador agents).
